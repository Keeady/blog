What is a heap?

Heap or binary heap is an array which can be represented as a binary tree where each node in the tree corresponds to an element in the array. The binary heap tree is filled up from left to right. The first element in the array is the root in the binary tree and the left child corresponds to 2 * i +1 and the right child corresponds to 2 * i + 2. Where i corresponds to the element index in the array.

For example: 16 is the root, 11 is left child of 16 and 9 is the right child. Similarly 11 is parent of 6 and 7.

                [16,11,9,6,7]

There are two types of binary heaps: max-heap and min-heap
