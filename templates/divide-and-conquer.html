Divide and Conquer By Example

Divide: Break down the problem into several smaller size sub-problems Conquer: Solve the sub-problems recursively Combine: combine the solutions from each sub-problems to create the final solution to the original problem

Merge Sort

Sorting an array of size N is our original problem. We know that sorting an array of size 1 or 2 is trivial. We can therefore break the task of sorting an array of any length by sorting each portion of the array at a time and merge the sorted parts together until we get a final sorted array

Divide: divide in half recursively until sorting each half is trivial. For instance, each half is of size 1 or 2 Conquer: Sort the portions Combine: Merge the sorted portions
